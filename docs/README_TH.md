## 🎯 Flux คืออะไร?

> **Flux ไม่ใช่แอป VPN สำหรับผู้ใช้ทั่วไป แต่เป็นโซลูชันโอเพนซอร์สสำหรับผู้ให้บริการ VPN ในการสร้างไคลเอนต์แบรนด์ของตัวเอง**

## ⚡ เริ่มใน 5 นาที

```bash
git clone https://github.com/flux-apphub/flux.git
cd flux
flutter pub get
# เปลี่ยน API URL ใน lib/services/api_config.dart
flutter run
```
