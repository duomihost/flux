## 🎯 Flux とは？

> **Flux は一般ユーザー向けの VPN アプリではありません。サービス事業者が独自クライアントを構築するためのオープンソースソリューションです。**

---

## 👥 誰のための Flux？

| ユーザータイプ | ニーズ | Flux でできること |

## ⚡ 5分で始める

```bash
git clone https://github.com/flux-apphub/flux.git
cd flux
flutter pub get
# lib/services/api_config.dart の API URL を変更
flutter run
```
